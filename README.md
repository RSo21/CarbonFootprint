# CarbonFootprint Calculator

### General info

An application that allows you to calculate your Carbon Footprint.


### Technologies:

Project is created with:

$ HTML5
$ SASS
$ React

### Setup

To run this project, install it locally using npm:

1  ```git clone https://github.com/RSo21/CarbonFootprint.git``` to clone the repository
2 ```npm install``` to install all dependencies
3 ```npm start``` and ```http://localhost:3000/CarbonFootprint`` in the browser to preview the app
